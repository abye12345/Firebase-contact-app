## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository:

```bash
git clone <https://github.com/abye12345/Firebase-Contact-App.git>
```

2. Navigate to the project directory:

```bash
cd my_fiber_app
```

3. Install dependencies:

```bash
npm install
# or
yarn install
```

4. Start the development server:

```bash
npm start
# or
yarn start
```

5. Open [http://localhost:3000](https://abye12345.github.io/Firebase-contact-app/) to view it in your browser.

## Usage

1. **Adding a Contact**

   - Click the '+' button in the top right
   - Fill in the name and email
   - Click 'Add Contact'

2. **Updating a Contact**

   - Click the pencil icon on any contact
   - Modify the information
   - Click 'Update Contact'

3. **Deleting a Contact**
   - Click the trash icon on any contact
   - The contact will be removed immediately
